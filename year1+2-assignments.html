NOTES:
ICMP (Internet Control Message Protocol)
is an error-reporting protocol network devices
like routers use to generate error messages to
the source IP address when network problems
prevent delivery of IP packets.

(arp)
Address Resolution Protocol (arp)
The address resolution protocol (arp)
is a protocol used by the Internet Protocol (IP)

Spanning Tree Protocol (STP) is a Layer 2 protocol that runs on bridges and switches.
The specification for STP is IEEE 802.1D. The main purpose of STP is to ensure that
you do not create loops when you have redundant paths in your network.
Loops are deadly to a network.
